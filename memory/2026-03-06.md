# 2026-03-06: Git LFS Migration

## Summary
Migrated 13,652 email HTML files (~883 MB) from regular git objects to Git LFS. Full history rewrite (3 commits). Updated CI workflow with LFS checkout + caching. Updated docs.

## Changes
- `git lfs migrate import --include="emails/**/*.html" --everything` rewrote all 3 commits
- `.gitattributes` auto-created: `emails/**/*.html filter=lfs diff=lfs merge=lfs -text`
- `.github/workflows/deploy.yml`: added `lfs: true`, LFS cache step, `git lfs pull` step
- `README.md`: added Git LFS prerequisite section
- `ARCHITECTURE.md`: documented LFS storage decision
- Git pack size: 137 MB -> 4 MB (LFS store: 882 MB)

## Notes
- `backup-before-lfs` branch created as safety net (points to same rewritten commits)
- Stale `refs/remotes/origin/main` was removed to allow GC to reclaim old objects
- `data/index.json` intentionally NOT in LFS (benefits from delta compression + diffing)

# 2026-03-06: Read Tracking & Bookmarks

## Summary
Added client-side read tracking and bookmarks feature using localStorage. No backend needed — all state persists in the browser.

## Changes
- `templates/app.js`: Added `Store` module (localStorage-backed with in-memory Set cache), bookmark/read state rendering across all pages, new `initBookmarks()` function
- `templates/style.css`: Read state dimming, bookmark indicators, active bookmark button, newsletter label pill, header bookmarks link + badge
- `templates/index.html`: Added "Bookmarks" link with badge in header
- `templates/view.html`: Replaced spacer div with bookmark toggle button
- `templates/bookmarks.html`: New page — lists bookmarked emails with search, newsletter labels
- Build script unchanged (already copies all .html from templates/)

## localStorage Schema
- `nl_read`: JSON array of file paths for read emails
- `nl_bookmarks`: JSON array of file paths for bookmarked emails

## Features
- Viewer auto-marks emails as read on open
- Bookmark toggle in viewer header (filled/outline icon)
- Read emails dimmed in newsletter email list
- Bookmark icon shown on bookmarked items in lists
- Read count ("X / Y read") on homepage cards
- Bookmarks badge count in homepage header
- Dedicated bookmarks page with search + newsletter labels
- Read toggle button (eye icon) in viewer — allows marking emails as unread after auto-read
- `Store.toggleRead()` method mirrors `toggleBookmark()` pattern
- Eye icons: open eye = read, closed eye with slash = unread
