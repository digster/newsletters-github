# 2026-03-07 — Inline Action Buttons on Email List Rows

## What Changed
- Added interactive read toggle (eye icon) and bookmark toggle buttons directly on email list rows
- Previously these controls only existed on the email viewer page; list items had static indicators only

## Files Modified
- `templates/app.js` — Refactored `renderEmailList()` to emit new HTML structure with `<button>` elements; added `bindListActions()` using event delegation pattern
- `templates/style.css` — Added `.email-item__content` wrapper, `.email-item__actions` container, `.email-item__action-btn` styles; removed old `.email-item__bookmark` static indicator; updated mobile breakpoint to stack content not the whole row
- `ARCHITECTURE.md` — Documented event delegation pattern for list-level actions

## Key Patterns
- Event delegation: one listener on container, `e.target.closest("[data-action]")` to find button
- `_actionsListenerBound` flag prevents duplicate listeners on re-renders
- `e.preventDefault()` + `e.stopPropagation()` blocks `<a>` navigation when clicking buttons
- 28x28px buttons, `var(--fg-faint)` default color, `var(--accent)` when active
